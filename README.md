# 🌿 Nature-Themed Personal Portfolio

A responsive and modern personal portfolio website built using HTML, CSS, and JavaScript. The portfolio features a beautiful nature-inspired design, smooth navigation, interactive elements, and a clean user interface.

## ✨ Features

- 🌿 Nature-themed background design
- 📱 Fully responsive layout
- 🏠 Home, About, Skills, Projects, and Contact sections
- 🚀 Smooth scrolling navigation
- 💻 Typing text animation
- 🎨 Modern glassmorphism cards
- 📊 Skill progress bars
- 🌟 Hover effects and animations
- 📄 Resume download button
- 🔗 Social media links

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🌐 Deployment

You can deploy this portfolio using:

### GitHub Pages

1. Push code to GitHub
2. Open Repository Settings
3. Go to Pages
4. Select:
   - Branch: `main`
   - Folder: `/root`
5. Save

Your website will be available at:

```text
https://github.com/SoumiBandyopadhyay/CodeAlpha____Portfolio-Site
```

### Netlify

1. Login to Netlify
2. Click "Add New Site"
3. Drag and drop project folder
4. Deploy instantly
